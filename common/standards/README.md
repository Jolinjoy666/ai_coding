# standards

Reusable IC design and verification standards.

## Available documents

- `systemverilog_rtl_coding.md`: synthesizable SystemVerilog RTL coding style.
- `reset_and_clocking.md`: reset and clocking strategy guidance.
- `fsm_design.md`: finite state machine structure and review rules.
- `ready_valid_protocol.md`: ready/valid protocol semantics and pitfalls.
- `pipeline_timing.md`: pipeline and timing-convergence guidance.
- `cdc_rdc_guidelines.md`: clock and reset domain crossing guidance.
- `memory_register_file.md`: memory wrapper, SRAM, RAM, and register file guidance.
- `rtl_arithmetic_fixed_point.md`: bit width, signedness, fixed-point, rounding, saturation, and overflow guidance.
- `low_power_rtl.md`: RTL-level low-power guidance.
- `dft_rtl_guidelines.md`: DFT-aware RTL guidance.
- `assertion_verification_guidelines.md`: RTL assertion and verification guidance.
- `signoff_static_checks.md`: lint, CDC/RDC, STA, LEC, and waiver guidance.
