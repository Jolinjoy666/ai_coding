# checklists

Reusable review and signoff checklists.

## Available checklists

- `project_bootstrap_checklist.md`: project directory, metadata, raw spec preservation, and common asset link checklist.
- `spec_readiness_checklist.md`: spec completeness, assumptions, constraints, and verification-intent checklist.
- `architecture_review_checklist.md`: architecture option and decision review checklist.
- `micro_arch_review_checklist.md`: micro-architecture implementation readiness checklist.
- `rtl_review_checklist.md`: RTL interface, logic, datapath, memory, CDC/RDC, PPA, and verification review checklist.
- `cdc_rdc_review_checklist.md`: CDC/RDC domain mapping, crossing classification, unsafe pattern, constraint, waiver, and verification checklist.
- `ppa_review_checklist.md`: performance, power, area, functional-safety, and validation checklist.
- `uvm_environment_review_checklist.md`: UVM agent/env/scoreboard/coverage readiness checklist.
- `regression_closure_checklist.md`: regression failure triage and closure checklist.
- `signoff_readiness_checklist.md`: final RTL, verification, static-check, PPA, waiver, and release-package checklist.
- `vcs_flow_review_checklist.md`: VCS environment, filelist, compile, run, wave, log-check, clean, and debug readiness checklist.
