# templates

Reusable templates for IC design projects.

## Available templates

- `project_init/`: project bootstrap templates for `project.yaml`, README, AI context, raw spec, and directory manifest.
- `simulation/vcs/`: Synopsys VCS Makefile, filelist, and log-check templates.

## Recommended template groups to add later

- RTL module templates.
- SystemVerilog package templates.
- Interface templates.
- UVM component templates.
- Testplan templates.
- Filelist templates.
- Makefile templates.
- Report templates.
