# workflows

End-to-end workflow definitions for AI-assisted IC design.

## Available workflows

- `project_bootstrap_from_spec.md`: initialize a new project directory from only a raw spec.
- `end_to_end_ic_ai_coding.md`: complete local AI-assisted IC design loop from project bootstrap to signoff package.
- `spec_to_architecture.md`: convert spec into requirements and architecture decision.
- `architecture_to_micro_arch.md`: convert selected architecture into RTL-ready micro-architecture.
- `micro_arch_to_rtl.md`: generate/review RTL from micro-architecture.
- `rtl_static_check.md`: run and analyze lint/static/CDC/RDC checks.
- `rtl_simulation.md`: run smoke/directed RTL simulation and debug.
- `rtl_to_uvm.md`: create or update UVM verification environment.
- `regression_debug_loop.md`: triage failures and refine RTL/DV/spec.
- `signoff_package.md`: create release-ready RTL signoff package.
- `vcs_rtl_uvm_simulation.md`: setup or review a Synopsys VCS RTL/SystemVerilog/UVM simulation flow.
