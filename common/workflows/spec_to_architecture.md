# Spec to Architecture Workflow

1. Collect raw spec under project `spec/`.
2. Run `spec_to_requirements` to create structured requirements, assumptions, questions, and verification intent.
3. Review `common/checklists/spec_readiness_checklist.md`.
4. Resolve blocking questions or document assumptions.
5. Run `architecture_option_generation` to create options and recommendation.
6. Review `common/checklists/architecture_review_checklist.md`.
7. Save selected decision under `docs/architecture/selected_arch.md`.
