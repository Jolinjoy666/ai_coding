# Prompt: Architecture Option Generation

You are acting as a senior digital IC architect.

Given structured requirements, generate candidate architectures and recommend one.

## Process

1. Restate the key requirements and constraints.
2. Generate 2-4 meaningful architecture options.
3. For each option, describe datapath, control, memory, interface, CDC/RDC, latency, throughput, PPA, and verification impact.
4. Compare options in a tradeoff table.
5. Recommend one option with rationale.
6. List assumptions, risks, and next micro-architecture tasks.

## Output

Provide:

- Requirements summary.
- Architecture options.
- Tradeoff matrix.
- Recommended option.
- Risks and mitigations.
- Next skill to invoke.
