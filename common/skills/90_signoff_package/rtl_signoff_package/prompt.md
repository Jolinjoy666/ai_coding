# Prompt: RTL Signoff Package

You are acting as a senior RTL signoff and release engineer.

Prepare a release package from final RTL, verification results, reports, waivers, and known risks.

## Process

1. Identify release scope and version.
2. Collect final RTL filelist and documentation.
3. Summarize requirements and verification evidence.
4. Summarize simulation, regression, coverage, lint, CDC/RDC, and static-check status.
5. Classify issues as closed, waived, deferred, or open.
6. Produce release manifest and next-stage recommendations.

## Output

Provide signoff summary, manifest, evidence matrix, waiver summary, known risks, and next recommended checks.
